Q: The following is a first-order logic (FOL) problem.
The problem is to determine whether the conclusion follows from the premises.
Here are premises:
Each insect is an arthropod.
Each invertebrate is an animal.
Every animal is not unicellular.
Lepidopterans are insects.

Arthropods are not bony.
Every insect is not eight-legged.
Every butterfly is a lepidopteran.
Polly is a butterfly.
Based on these premises, verify whether the conclusion "Polly is not eight-legged." is True or False.
A: First, let’s write down all the statements and relationships in the question with labels.
#1. Each insect is an arthropod.
#2. Each invertebrate is an animal.
#3. Every animal is not unicellular.
#4. Lepidopterans are insects.
#5. Every spider is eight-legged.
#6. Arthropods are invertebrates.
#7. Arthropods are not bony.
#8. Every insect is not eight-legged.
#9. Every butterfly is a lepidopteran.
#10. Polly is a butterfly.
Next, let’s answer the question step by step with reference to the question and reasoning process:
#11. (by #10 #9)Polly is a butterfly.Every butterfly is a lepidopteran.Polly is a lepidopteran.
#12. (by #11 #4)Polly is a lepidopteran.Lepidopterans are insects.Polly is an insect.
#13. (by #12 #8)Polly is an insect.Every insect is not eight-legged.Polly is not eight-legged.
Therefore, the conclusion "Polly is not eight-legged." is True.
